Title

概要

これはHTML+JavaScript+CSSで作成した, HTMLエディタです. 任意の数のテキストセクション, 画像ファイルの追加や, ハイパーリンクの設定が行えることが特徴となっています. 

このアプリケーションはローカルサーバを用意し, その上で使用することを前提しています. 

使い方

編集モードに入るには左上の鉛筆ボタンを押します. 編集を終える際にはその隣のチェックマークを押します. ページを保存するには, フロッピーディスクボタンを押します.

「+section」ボタンを押すとテキストセクションが追加されます. 一つのセクションはタイトル部分と, パラグラフ群から構成されます.
